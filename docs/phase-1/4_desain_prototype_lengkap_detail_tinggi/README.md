# Prototipe High-Fidelity (Detail Tinggi)
## Janasku RAG Chatbot

Folder ini berisi kode murni (Vanilla HTML, CSS, dan JavaScript) untuk prototipe *High-Fidelity* dari aplikasi *chatbot* Janasku.

### Cara Menjalankan
Karena ini menggunakan HTML/CSS/JS murni tanpa *framework*, Anda cukup membuka file `index.html` langsung di *browser* Anda (klik ganda file `index.html` atau *drag* ke Chrome/Safari).

### Fitur Desain
1. **Estetika Premium Organik**: Menggunakan palet warna yang terinspirasi dari bahan baku Janasku: Kunyit (Amber/Oranye), Jahe (Kuning Lembut), dan *background blur* (Glassmorphism).
2. **Animasi Latar Belakang**: Blob warna yang melayang halus memberikan kesan hidup dan modern pada antarmuka.
3. **Layar Ganda (Split-Screen)**:
   - **Kiri**: Panel *Knowledge Brain* untuk menyimulasikan tempat admin menyeret dan melepas (*drag & drop*) file SOP/Harga.
   - **Kanan**: Panel Simulasi Chatbot untuk mengetes jawaban AI secara langsung.

### Interaksi JavaScript (Mokup)
File `app.js` menyediakan simulasi logika agar prototipe terasa hidup:
- Anda dapat menyeret file apapun ke zona *Upload* (akan muncul daftar file di bawahnya).
- Anda dapat mengetik pesan di kolom *chat*, dan aplikasi akan memunculkan simulasi balasan *bot* beserta *typing indicator*.

# High-Fidelity Prototype
## Janasku RAG Chatbot

This folder contains pure, vanilla code (HTML, CSS, and JavaScript) for the High-Fidelity prototype of the Janasku chatbot application.

### How to Run
Since this utilizes pure HTML/CSS/JS without any build framework, you can simply open the `index.html` file directly in your browser (double-click the `index.html` file or drag it into Chrome/Safari).

### Design Features
1. **Premium Organic Aesthetics**: Utilizes a color palette inspired by Janasku's raw ingredients: Turmeric (Amber/Orange), Ginger (Soft Yellow), and Glassmorphism (background blurs).
2. **Background Animations**: Smooth, floating color blobs provide a lively, modern, and engaging interface.
3. **Split-Screen Layout**:
   - **Left**: *Knowledge Brain* panel simulating where an admin can drag & drop SOP or Pricing files.
   - **Right**: Chatbot Simulation panel to test AI responses directly.

### Mockup JavaScript Interactions
The `app.js` file provides simulated logic to make the prototype feel alive:
- You can drag any file into the Upload zone (it will populate the active documents list below it).
- You can type a message in the chat input, and the application will simulate a bot reply complete with a typing indicator.
